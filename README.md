# Assistente-financeiro-iaAssistente Financeiro com IA Simples

Projeto em Python que simula uma experiência digital de relacionamento financeiro com respostas inteligentes, contexto da conversa, FAQ bancário e cálculos financeiros demonstrativos.

Funcionalidades
Conversa em linguagem natural
FAQ sobre PIX, cartão, empréstimo, investimento e segurança
Simulação de investimento
Simulação de empréstimo
Persistência de contexto durante a sessão
Histórico de perguntas do usuário
Respostas claras e personalizadas
